# 🔍 Buró de Auditoría Financiera 2026

## Descripción del Proyecto

Sitio web corporativo profesional, académico e interactivo que representa un Buró de Auditoría Financiera completo. Desarrollado con **HTML5, CSS3 y JavaScript Vanilla** (sin frameworks externos) para máxima compatibilidad y rendimiento.

---

## 📋 Estructura del Proyecto

### Archivos HTML

1. **index.html** - Página Principal
   - Hero section con llamada a la acción
   - Presentación del equipo auditor (4 integrantes)
   - Indicadores animados del proyecto
   - Características principales del buró
   - Sección CTA

2. **proyecto.html** - Presentación del Proyecto
   - Introducción editable
   - Objetivo general destacado
   - Objetivos específicos con iconos
   - Importancia de la auditoría (6 aspectos)
   - Metodología aplicada
   - Documentación del proyecto

3. **firma.html** - Firma Auditora
   - Información general de la firma
   - Misión y visión institucionales
   - Valores corporativos (6 valores)
   - Organigrama visual interactivo
   - Áreas de competencia
   - Certificaciones y acreditaciones
   - Experiencia y trayectoria

4. **empresa.html** - Empresa Auditada
   - Información general de la entidad
   - Antecedentes históricos
   - Actividad económica principal
   - Estructura organizacional
   - Áreas evaluadas (6 áreas)
   - Indicadores de desempeño
   - Evaluación de riesgos
   - Resumen ejecutivo

5. **desarrollo.html** - Desarrollo del Buró (Más Importante)
   - Dashboard de auditoría con estadísticas
   - 6 procedimientos completos con:
     - Procedimiento aplicado
     - Evidencia obtenida
     - Hallazgo
     - Recomendación
     - Conclusión
   - Tabs interactivas (Procedimientos, Hallazgos, Recomendaciones)
   - 5 Hallazgos clasificados por prioridad
   - 6 Recomendaciones priorizadas
   - Línea de tiempo interactiva (7 fases)
   - Métricas finales
   - Conclusiones generales

## 🔧 Uso

### Abrir el Sitio
1. Descargar/clonar los archivos
2. Abrir `index.html` en navegador
3. O servir los archivos desde un servidor

### Desde Servidor Local (Recomendado)
```bash
# Si tienes Python 3
python -m http.server 8000

# Si tienes Python 2
python -m SimpleHTTPServer 8000

# Si tienes Node.js
npx http-server
```

Luego abrir: `http://localhost:8000`
